# 🏏 IPL Live Win Predictor & Analytics

This project is a web application that analyzes IPL match data and predicts the live winning probability of a team based on match conditions.

The application is built using Python and deployed using Streamlit Cloud.

Live App:
https://iplpredictor-aki.streamlit.app/

---

## Project Description

The main objective of this project is to analyze historical IPL data and build a machine learning model that can predict the probability of a team winning during a live match situation.

Users can explore team statistics and also enter match details such as:

- Batting team
- Bowling team
- Venue
- Runs scored
- Overs completed
- Wickets fallen

Based on these inputs, the model calculates the winning probability.
